# @aleph-fi/widget-react

React component for Aleph Widget.

## Installation

```bash
npm install @aleph-fi/widget-react
```

## Usage

```tsx
import { AlephWidget } from '@aleph-fi/widget-react';

function App() {
  return (
    <div>
      <AlephWidget />
    </div>
  );
}
```

## Props

| Prop | Type | Description |
|------|------|-------------|
| `className` | `string` | Custom className for the widget container |
| `style` | `React.CSSProperties` | Additional styles for the widget |
